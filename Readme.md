# Manage-X

Manage-X is a powerful application designed to help you manage your tasks efficiently. It comes with a variety of features including task creation, task assignment, task tracking, and reporting.

## Local Setup

1. Clone the repository to your local machine using the following command: `git clone https://github.com/username/Manage-X.git`
2. Navigate to the root directory of the repository: `cd Manage-X`
3. Install the dependencies using the following command: `pip install -r requirements.txt`
4. Start the application using the following command: `python manage.py runserver`

## Cloud Hosting Setup

1. Create a new instance on your preferred cloud hosting service.
2. Deploy the application to the instance following the instructions provided by the hosting service.
3. Start the application following the instructions provided by the hosting service.

## Dependencies

- Django: A high-level Python Web framework that encourages rapid development and clean, pragmatic design.
- PostgreSQL: A powerful, open source object-relational database system.
- Gunicorn: A Python WSGI HTTP Server for UNIX.
- Whitenoise: Simplified static file serving for Python web apps.
